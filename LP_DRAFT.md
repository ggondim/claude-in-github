# autoducks
> Run any code agent, anywhere, with any provider.

- Agents: execute tasks and workflows. Templates include: `Task Execution Agent`, `Spec Design Agent`, `Tactical Planner`, `Wave Orchestrator`.
- ITS: integrate with your existing issue tracking system and trigger agentic workflows. Currently supports: `github-issues`.
- Providers: connect to various LLMs and tools. Currently supports: `claude-github-actions-oauth`.
- Runtimes: execute agents in different environments. Currently supports: `github-actions`.
